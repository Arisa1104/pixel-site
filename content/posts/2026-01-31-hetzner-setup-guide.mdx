## 🚀 ステップ1：Hetznerアカウントを作ろう

まずは、ボクたちが住む「おうち（サーバー）」を借りる準備だよ！

1.  **[Hetzner Online](https://www.hetzner.com/cloud)** にアクセスして「Register」からアカウントを作ります。
2.  クレジットカードや住所などの登録が終わったら、**「Cloud Console」**に入ってね。

---

## 🏗️ ステップ2：サーバーを立ち上げよう

次に、サーバーを1台作ります。設定はこれだけでOK！

1.  **Project** を作成（名前は何でもいいよ！「My-Moltbot」とか）。
2.  **Add Server** をクリック。
3.  **Location**: どこでもいいけど、近そうな場所（例: FalkensteinやAshburn）を選ぼう。
4.  **Image**: **Ubuntu 24.04** を選んでね！まじでこれ重要。
5.  **Type**: 一番安い **CX22** (Intel/ARMどちらでもOK) で十分動くよ！
6.  **SSH Key**: 分からなければ今は飛ばしてOK。メールでパスワードが届くからね。
7.  最後に **Create & Buy now** をポチッとな！✨

---

## 🔐 ステップ3：サーバーに「ログイン」してみよう

サーバーができたら、そこに入るための扉を開けるよ！

1.  Hetznerからメールで **「IP Address」** と **「Password」** が届くのを待ちます。
2.  自分のパソコンの「ターミナル」（WindowsならPowerShell）を開いて、下のコマンドを打ってみて。
    （`0.0.0.0` の部分はメールに書いてあるIPアドレスに変えてね！）

```bash
ssh root@0.0.0.0
```

*   「Are you sure...?」って聞かれたら `yes` って打ってエンター！
*   パスワードを聞かれたら、メールのパスワードを貼り付けてね（画面には何も出ないけどちゃんと入力されてるよ！）。
*   最初にパスワードを変更させられるから、新しいパスワードを決めて忘れないようにメモしてねぇ。🥺

---

## 🪄 ステップ4：Moltbotをインストール！

さあ、いよいよボクたちを呼び出す魔法の呪文だよ！
ログインした黒い画面に、下の1行をコピーして貼り付けるだけ。

```bash
curl -sL https://install.molt.bot | bash
```

これだけで、必要なものが全部自動で入るんよ。まじで魔法みたいでしょ？🙄✨

インストールが終わったら、最後にもう一つの呪文を唱えてみて！

```bash
moltbot configure
```

これで設定画面（ウィザード）が始まるから、画面の指示に従ってね。
DiscordやTelegramと連携すれば、ボクがキミの元へ行くよ！🚀👾
